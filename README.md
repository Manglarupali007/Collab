# 💬 Nexus - Real-Time Chat Application

Nexus is a modern real-time chat application that enables users to communicate instantly in private rooms. It supports secure authentication, live messaging, file sharing, reactions, polls, task management, and real-time collaboration using Socket.IO.

## ✨ Features

* 🔐 Secure User Authentication (JWT)
* 💬 Real-Time Messaging
* 👥 Private Chat Rooms
* 📎 File & Image Sharing
* 😀 Emoji Reactions
* 📌 Pin & Unpin Messages
* 📊 Poll Creation & Voting
* ✅ Task Management
* ⚡ Socket.IO Powered Live Communication

## 🛠 Tech Stack

* React.js
* Node.js
* Express.js
* Socket.IO
* MongoDB
* JWT Authentication
* Vercel (Frontend)
* Render (Backend)
